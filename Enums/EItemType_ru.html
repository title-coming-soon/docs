Enum EItemType

EItemType определяет тип прдемета.

UENUM(Blueprintype)
enum class EItemType : uint8
{
Item,
WeaponRanged,
WeaponMelee,
Gadget,
Alteration,
Deco,
Ammo,
Token,
Quest,
QuestBundle,
NeverPersist,
}

ПОМЕТКА: Это не полный список всех возможных элементов энумиратора, это только базовый набор и каждая игра имеет свои доплнительные элементы.
