ITEM DEFINITION

Item Definition это базовый класс для всех предметов. Инвентарь играет очень большую роль в наших играх, так как через него проходит больше, чем просто предметы игрока: большое количество проверок и так же инвентарь используется системой способностей.

Бпзовый набор перменных в классе UItemDefinition.

ТипНазваниеОписание
FStringItemIdУникальный Id предмета в виде строки.
FTextDisplayNameОтображаемое название предмета.
